# KSA Times

A digital archive and reader for KSA Times magazines.

## Features

- **Magazine Archive**: Browse past issues of KSA Times.
- **PDF Reader**: Integrated flipbook-style PDF reader for an immersive reading experience.
- **Admin Interface**: Secure upload and management of magazine issues.
- **Responsive Design**: Optimized for both desktop and mobile viewing.

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Vite
- **Backend**: Express.js (serving static files and API)
- **PDF Handling**: react-pdf, react-pageflip

## Development

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```
